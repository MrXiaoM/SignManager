# SignManager

mirai 签名服务管理器/安装器。

# 使用方法

本软件仅支持在 Windows 下运行，你可以在 Windows 下配置好签名服务再迁移至其他系统使用。

欲运行本程序，需要安装 [.NET Core 6.0](https://dotnet.microsoft.com/zh-cn/download/dotnet/6.0)。
